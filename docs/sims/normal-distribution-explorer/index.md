## About This MicroSim

Explore how the **mean (μ)** and **standard deviation (σ)** parameters shape a normal distribution curve.

### How to Use
- Adjust **μ₁ slider** to shift the curve left or right (changes center)
- Adjust **σ₁ slider** to make the curve wider/flatter or narrower/taller
- Click **Compare** to add a second curve for side-by-side comparison
- Use **Reset** to return to default values

### Key Insights
- **Mean (μ)** determines the center/peak location of the curve
- **Standard deviation (σ)** determines the spread/width
- Larger σ → wider, shorter curve (more spread out)
- Smaller σ → narrower, taller curve (more concentrated)
- Total area under any normal curve always equals 1

## Lesson Plan

### Learning Objective
Students will predict how changes to μ and σ will affect the shape and position of a normal distribution curve (Bloom's Taxonomy: Understanding, Analyzing).
